# 《虚空塔》游戏策划书

## 一、游戏概述

- **游戏名称**：虚空塔（Void Tower）
- **游戏类型**：卡牌构筑 Roguelike（Deckbuilding Roguelike）
- **目标平台**：PC 
- **开发引擎**：Godot 4.x
- **参考作品**：《Slay the Spire》《杀戮尖塔2》《Monster Train》《Inscryption》
- **核心体验**：每一次爬塔都是一次全新的构筑之旅——在卡牌、印记、虚空裂痕的三重抉择中，找到属于你的那套「不可能的组合」，用一套自洽的构筑碾碎敌人。

---

## 二、核心创意点（5大创新）

### 创意一：双色能量系统（红蓝能量）

打破传统单能量体系，引入 **体力（红）** 与 **精神力（蓝）** 双能量。

| 能量类型 | 来源 | 用途 | 残留规则 |
|---------|------|------|---------|
| 体力（红） | 每回合自动回满 | 物理攻击、防御、移动 | 回合结束清零 |
| 精神力（蓝） | 每回合固定获得 | 法术、召唤、改造卡牌 | 可跨回合累积，上限为精神力槽 |

**为什么有趣？**
- 物理系卡牌依赖当回合爆发，法术系卡牌可以蓄力打 combo
- 红蓝混伤流派需要精密计算两种资源节奏
- 敌人可以攻击你的精神力槽，迫使法术流派改变打法

---

### 创意二：卡牌刻印系统（Rune Inscription）

每张卡牌拥有 **1-2 个刻印槽**，在地图节点「符文熔炉」中，可以将符文镶嵌到卡牌上。

```
┌──────────────────┐
│  「烈焰斩」       │
│  造成 12 点伤害   │
│  ┌──────────┐    │    镶嵌后 →
│  │ 槽1: 分裂  │    │
│  │ 槽2: 空   │    │
│  └──────────┘    │
└──────────────────┘

┌──────────────────┐
│  「烈焰斩·分裂」  │
│  造成 12 点伤害   │
│  → 同时命中所有敌人│
│  ┌──────────┐    │
│  │ 槽1: 分裂  │    │
│  │ 槽2: 空   │    │
│  └──────────┘    │
└──────────────────┘
```

**符文类型示例：**

| 符文 | 效果 |
|------|------|
| 分裂 | 单体 → 全体 |
| 回响 | 使用后复制一张临时副本到手牌 |
| 延时 | 伤害延迟1回合，但伤害×1.5 |
| 嗜血 | 消耗2点生命，伤害+4 |
| 灵体 | 此卡不再消耗能量，改为消耗等量精神力 |
| 稳固 | 此卡不会被敌人的「弃置」或「削除」效果影响 |

**玩法意义**：同一张卡在不同刻印组合下产生完全不同的构筑方向，极大提升卡池深度。

---

### 创意三：虚空裂隙系统——主动风险/回报机制

不同于传统 Roguelike 中被动接受随机事件，**虚空裂隙**是玩家主动选择进入的高风险空间。

**机制**：每层地图上散布 1-2 个不稳定裂隙，进入后触发一个 **扭曲规则 + 精英战**：

| 裂隙名称 | 扭曲规则 | 奖励 |
|---------|---------|------|
| 逆流裂隙 | 手牌顺序反转，从右向左打出 | 双倍金币 |
| 静默裂隙 | 本场战斗不能使用技能牌 | 获得一张稀有卡 |
| 镜像裂隙 | 敌人复制你每回合打出的第一张牌 | 获得一个传奇符文 |
| 枯竭裂隙 | 每回合少抽1张牌 | 随机升级3张牌 |
| 混沌裂隙 | 每回合能量随机 ±1 | 获得一件传说遗物 |

**设计意图**：
- 给高水平玩家提供「秀操作」的空间
- 连败时可以通过冒险搏翻盘机会
- 扭曲规则本身也可以成为构筑的试金石——能抗住静默裂隙的卡组才是好卡组

---

### 创意四：卡牌进化树

卡牌不再是「基础版 → 升级版」的二元升级，而是拥有 **2-3 条分支进化路径**。

```
              ┌──→ 「陨石术」  高伤害，自身扣血
              │
「火球术」 ───┼──→ 「连珠火球」 中等伤害，打3次
              │
              └──→ 「不灭之火」 低伤害，但使用后返回牌库
```

**升级节点**：在「贤者祭坛」节点花费金币选择一条进化路径。进化后的卡牌解锁新的刻印槽。

**设计意图**：减少「固定最优解」——根据当前构筑的遗物和符文，不同进化路径各有价值。

---

### 创意五：塔灵羁绊系统

选择角色时同时选择一位 **塔灵同伴**，塔灵不直接参战，但提供全局被动 + 一个主动技能（每场战斗限1次）。

| 塔灵 | 被动 | 主动技（每战1次） |
|------|------|------------------|
| 灰烬之鸦 | 每丢弃一张牌，回复1点生命 | 弃掉手牌中所有非攻击牌，每张抽1张 |
| 水晶蝶 | 每回合第一张法术牌费用-1 | 本回合精神力消耗减半 |
| 暗影蛇 | 敌人每有一种负面状态，受到伤害+1 | 将一张「毒液」洗入敌人牌库 |
| 铸铁龟 | 每场战斗首次受到伤害-3 | 获得等于当前护甲的临时力量 |
| 时空猫 | 每回合可以重抽1张手牌 | 本回合额外获得2点能量 |

**设计意图**：塔灵是你的「第 0 张牌」——围绕塔灵构建卡组，增加开局多样性和角色粘性。

## 三、游戏系统设计

### 3.1 角色设计

#### 初始角色：虚空骑士（新手友好）

- **初始生命**：70 → 升级 +5
- **体力（红）**：3/回合
- **精神力（蓝）**：上限 4，获得 1/回合
- **初始卡组**：5 攻击 / 4 防御 / 1 技能
- **初始遗物**：「骑士誓言」——每场战斗首回合获得 2 点临时护甲
- **构筑方向**：均衡型，红蓝兼修

#### 解锁角色：魂火术士

- **初始生命**：55
- **体力（红）**：2/回合
- **精神力（蓝）**：上限 8，获得 2/回合
- **被动**：精神力溢出时（超过上限），溢出部分转化为护甲
- **构筑方向**：精神力特化，蓄力爆发

#### 解锁角色：碎刃刺客

- **初始生命**：60
- **体力（红）**：4/回合
- **精神力（蓝）**：上限 2，获得 1/回合
- **被动**：同一回合打出第 4 张攻击牌时，那张牌伤害翻倍
- **构筑方向**：体力特化，连击 combo

#### 隐藏角色：虚空之嗣（击败最终 BOSS 解锁）

- **初始生命**：40
- **能量**：无红蓝之分，统一为 5 点虚空能量
- **被动**：每场战斗可以将一张手牌「虚空化」——永久移除但获得其伤害/护甲数值 1/2 的永久属性加成
- **构筑方向**：极限精简卡组

### 3.2 地图结构

```
                     ┌──────┐
                     │ BOSS │
                     └──▲───┘
                  ┌──────┴──────┐
                  │   精英      │
                  └──▲───▲─────┘
              ┌──────┘   └──────┐
         ┌────┴───┐         ┌───┴────┐
         │ 符文   │         │ 虚空   │
         │ 熔炉   │         │ 裂隙   │
         └──▲──▲──┘         └───▲────┘
      ┌──────┘  └──────┐       │
  ┌───┴───┐        ┌───┴───┐   │
  │ 战斗   │        │ 贤者   │   │
  │       │        │ 祭坛   │   │
  └──▲──▲─┘        └───▲───┘   │
     │  └─────────┐    │       │
  ┌──┴───┐    ┌───┴────┴───┐   │
  │ 休息  │    │   商店     │   │
  └──▲──▲─┘   └──▲──▲──────┘   │
     │  │        │  │          │
  ┌──┴──┴───┐    │  │    ┌─────┴──┐
  │  起始    │    │  │    │ 起始   │
  └─────────┘    └──┴────┘────────┘
```

**节点类型**：
| 节点 | 说明 |
|------|------|
| 战斗 | 普通遭遇战 |
| 精英 | 强敌，掉落遗物 |
| 休息 | 回复 30% HP，或升级 1 张牌 |
| 商店 | 买卡/遗物/符文/药水 |
| 符文熔炉 | 镶嵌或更换符文 |
| 贤者祭坛 | 卡牌进化（消耗金币） |
| 虚空裂隙 | 扭曲规则战 |
| 问号 | 随机事件 |
| BOSS | 层底首领 |

**总层数**：4 大层，每层 12-16 个节点，最终为 BOSS。地图分支路线可见，玩家自主规划路线。

---

### 3.3 遗物系统

遗物分 5 个稀有度，总计约 80+ 件：

| 稀有度 | 示例 |
|--------|------|
| 普通 | 「破旧护符」精神力上限+1 |
| 罕见 | 「红蓝转轮」每回合可将 1 点红能化为 1 点蓝能（或反之） |
| 稀有 | 「共鸣之石」每打出一张带有符文的牌，抽一张牌 |
| 传说 | 「虚空心脏」每场战斗可免死一次，触发后恢复30% HP |
| BOSS专属 | 「塔之主冠」战斗开始时随机进化手牌中的一张牌 |

---

### 3.4 敌人设计

每个大层有一个主题派系，每场战斗由 1-3 个敌人组成：

**第一层——锈蚀遗迹**
- **锈蚀哨兵**：攻击附带「锈蚀」debuff（下回合抽到的第一张牌费用+1）
- **遗迹石魔**：每 3 回合蓄力一次大范围 AOE
- **层底 BOSS：锈蚀巨像**——每损失 25% HP，召唤一个哨兵

**第二层——幽影回廊**
- **暗影信徒**：会偷取你的精神力
- **幽魂之主**：每回合复制你弃牌堆里的一张牌并打出
- **层底 BOSS：千面魔君**——每次受到攻击后改变弱点抗性

**第三层——元素熔炉**
- **炎晶核心**：死亡时爆炸，对双方造成伤害
- **冰霜女巫**：可以冻结你的刻印槽（冻结期间符文失效）
- **层底 BOSS：混沌双子**——两个 BOSS 共享生命池，每 2 回合交换位置和能力

**第四层——虚空之巅**
- **虚空裂片**：无视护甲造成百分比伤害
- **虚空守卫**：会删除你牌库顶部的牌
- **最终 BOSS：虚空之心**——四阶段变身 BOSS

---

### 3.5 进度与解锁

| 项目 | 解锁条件 |
|------|---------|
| 魂火术士 | 使用虚空骑士通关一次 |
| 碎刃刺客 | 单回合打出 8 张攻击牌 |
| 虚空之嗣 | 击败最终 BOSS |
| 塔灵：暗影蛇 | 给敌人叠加 50 层中毒 |
| 塔灵：时空猫 | 在商店消费累计 500 金币 |
| 每日挑战 | 首次通关后解锁 |
| 自定义模式 | 解锁全部角色 |

## 四、开发规划（Godot 实现要点）

### 4.1 项目架构建议

```
res://
├── src/
│   ├── core/           # 核心系统
│   │   ├── battle/     # 战斗逻辑（回合、能量、状态机）
│   │   ├── card/       # 卡牌数据与效果系统
│   │   ├── relic/      # 遗物系统
│   │   ├── rune/       # 刻印/符文系统
│   │   └── map/        # 地图生成与节点管理
│   ├── data/           # CSV/JSON 数据表
│   │   ├── cards/
│   │   ├── relics/
│   │   ├── runes/
│   │   ├── enemies/
│   │   └── events/
│   ├── ui/             # UI 场景
│   │   ├── battle/
│   │   ├── map/
│   │   ├── deck/
│   │   └── menu/
│   └── vfx/            # 特效
├── assets/             # 美术与音频
└── tests/              # 单元测试
```

### 4.2 关键技术点

1. **卡牌效果系统**：使用 Godot 的 `Callable` 或自定义 `Effect` 资源类，实现数据驱动的卡牌效果
2. **存档系统**：使用 Godot 的 `ResourceSaver` 序列化运行状态
3. **地图生成**：使用加权随机图算法生成有向无环图
4. **AI 系统**：使用行为树或基于意图的模式（敌人公开下一回合意图）
5. **多语言**：使用 Godot 内置的 CSV 翻译系统

### 4.3 开发阶段

| 阶段 | 内容 | 预计耗时 |
|------|------|---------|
| 原型期 | 核心战斗 + 30张卡牌 + 1个角色 + 1层地图 | 4-6 周 |
| Alpha | 全部 4 层 + BOSS + 2 角色 + 遗物系统 | 8-12 周 |
| Beta | 符文刻印 + 卡牌进化 + 塔灵 + 平衡调整 | 6-8 周 |
| 打磨 | UI/UX 优化、音效、多语言、Bug修复 | 4-6 周 |

## 五、竞品差异化总结

| 维度 | Slay the Spire | 本作 |
|------|---------------|------|
| 能量系统 | 单一能量 | 红蓝双能量 |
| 卡牌成长 | 升级一次（+） | 2-3 条进化分支 |
| 卡牌定制 | 固定效果 | 刻印符文改造 |
| 风险系统 | 被动事件 | 主动裂隙挑战 |
| 角色差异化 | 卡池不同 | 卡池 + 塔灵 + 能量配置 |
| 地图策略 | 路线选择 | 路线 + 裂隙风险博弈 |

---

## 六、核心理念

> **「每一张牌都有故事，每一次选择都有代价，每一套构筑都是独一无二的。」**

不是让你找到最强的牌，而是让你把一张普通的牌用出最强的效果。
